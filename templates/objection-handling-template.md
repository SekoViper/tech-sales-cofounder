# Objection Handling Template

| Objection | Real concern | Clarifying question | Response | Next step |
|---|---|---|---|---|
| It is expensive | Value unclear | Expensive compared to what? | Let us compare the cost to the cost of the problem. | Build ROI case |
| We use Excel | Current workaround feels enough | What does Excel fail to do today? | Excel stores data, but it does not automate follow-up or show live reports. | Demo workflow |
| Staff will not use it | Adoption risk | Which workflow is easiest to start with? | Start small and train around one workflow. | Pilot |
| Let me think | Hidden blocker | What part do you need to think through? | Identify price, timing, authority, or fit. | Schedule follow-up |
